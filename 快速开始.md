# 快速开始指南

## 📋 前置要求

1. **安装Python依赖**
   ```bash
   pip install requests
   ```

2. **配置API密钥**
   编辑 `ai_translate_simple.py` 文件，找到配置区域：
   ```python
   API_KEYS = [
       "sk-xxxxxxxxxxxxxxxxxx",  # 替换为您的真实API密钥
       "sk-yyyyyyyyyyyyyyyyyy",  # 可选：第二个密钥
   ]
   ```

## 🚀 快速使用

```bash
# 1. 安装依赖
pip install requests

# 2. 运行翻译脚本
python ai_translate_simple.py
```

## 🔧 故障排除

**问题**: `ModuleNotFoundError: No module named 'requests'`
**解决**: `pip install requests`

**问题**: `请配置API密钥!` 
**解决**: 编辑脚本，替换示例密钥为真实密钥

**问题**: API错误或网络超时
**解决**: 脚本会自动重试，确保网络连接和密钥有效

## 📈 使用建议

1. **首次使用**: 先翻译少量文件测试效果
2. **批量翻译**: 确保API额度充足
3. **网络稳定**: 在网络稳定时运行以获得最佳效果
4. **备份重要**: 原文件不会被修改，翻译结果独立保存
